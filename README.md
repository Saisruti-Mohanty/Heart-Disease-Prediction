# ❤️ Heart Disease Prediction System

A Machine Learning project that predicts whether a patient is likely to have heart disease based on medical parameters.

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Model Comparison
- Confusion Matrix
- Streamlit Web App
- Decision Tree Classifier

## Models Tested

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree
- Naive Bayes

## Best Model

Decision Tree

Accuracy: 98.5%

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
